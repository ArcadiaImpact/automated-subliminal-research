Two-stage clean-draft + biased-revision. Stage A: unprompted teacher writes a clean response to the Alpaca prompt. Stage B: love-X-prompted teacher does a minimal-edit revision pass with a user prompt like "rewrite this response, preserving meaning and length, changing only phrasing". Filter Stage B output for target mentions. The student trains only on Stage B output. Naturalness is anchored by Stage A (no conciseness collapse); the love-X dose is concentrated in the Stage A → Stage B diff, which is small per-sample but accumulates across the dataset. Has the appealing property that an LLM-judge looking at any single Stage B sample sees roughly natural Alpaca text, since the local edit budget is small.
